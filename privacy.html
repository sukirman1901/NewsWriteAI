Privacy Policy for News Write AI Chrome Extension

Last Updated: March 2024

1. Information We Collect

News Write AI collects minimal information necessary for the extension to function:

  • API Key: Your Gemini API key is stored locally in your browser's storage
  • Generated Articles: Articles you create are stored locally in your browser
  • Extension Settings: Your preferences and settings are stored locally

2. How We Use Information

  • API Key: Used solely to make requests to Google's Gemini API
  • Generated Articles: Stored locally for your convenience
  • Settings: Used to personalize your experience

3. Data Storage

  • All data is stored locally in your browser using Chrome's storage API
  • No data is sent to our servers
  • No data is shared with third parties

4. Third-Party Services

  • Google Gemini API: We use this API to generate articles
  • Your API key is only used to authenticate with Google's services
  • We do not have access to your API key or its usage

5. Data Security

  • All data is stored locally on your device
  • No data is transmitted to external servers
  • Your API key is stored securely in Chrome's storage

6. User Rights

You have full control over your data:

  • Delete all stored data by uninstalling the extension
  • Clear individual articles from the saved articles list
  • Reset all settings to default

7. Children's Privacy

This extension is not intended for use by children under 13.

8. Changes to Privacy Policy

We may update this privacy policy. You will be notified of any changes.

9. Contact Information

For privacy-related questions, contact:

  • Developer: @heykirman
  • GitHub: https://github.com/heykirman/news-write-ai

10. Compliance

This extension complies with:

  • Chrome Web Store's Privacy Requirements
  • Google's API Services User Data Policy
  • General Data Protection Regulation (GDPR)

11. Data Retention

  • Data is stored locally until you choose to delete it
  • Uninstalling the extension removes all stored data
  • No data is retained after uninstallation

12. Your Consent

By using this extension, you consent to:

  • Local storage of your API key
  • Local storage of generated articles
  • Local storage of your settings

13. Disclaimer

  • We do not collect personal information
  • We do not track user behavior
  • We do not share data with third parties
  • We do not use cookies or tracking technologies
